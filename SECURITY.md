# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Vibenalytics, please report it responsibly.

**Do not open a public issue.** Instead, email us at **hello@vibenalytics.dev** with:

- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will acknowledge your report within 48 hours and aim to provide a fix within 7 days for critical issues.

## Scope

This policy covers the Vibenalytics CLI (`vibenalytics`) and the [Claude Code plugin](https://github.com/vibenalytics/vibenalytics-claude-plugin).

For issues related to the hosted dashboard at [app.vibenalytics.dev](https://app.vibenalytics.dev), please use the same email address.
